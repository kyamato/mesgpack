## NULL Block
chunkの１レコードが終了したのち、先頭から NULL バイトが並んでいる場合、NULL バイトブロックのサイズを表示する。NULLバイトのバイト数とNULL バイトが出現した場所のオフセット(ファイル名を含む先頭のchunkヘッダが終わった場所からのオフセット)を表示している。
